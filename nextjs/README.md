# Next.js Project Starter

## Project includes

- eslint
- stylelint
- flow
- jest

## How to use

Download the starter:

```bash
curl https://codeload.github.com/hiromitz/starter/tar.gz/master | tar -xz --strip=1 starter-master/nextjs
cd nextjs
```

Install it and run dev

```bash
yarn install
yarn dev
```

Build and run

```bash
yarn build
yarn start
```
